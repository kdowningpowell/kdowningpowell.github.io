# kdowningpowell.github.io
This repository contains the source code for my personal portfolio website. The site showcases my technical skills, projects, resume, and professional background as a Computer Technology student and DevSecOps professional.

The portfolio was designed to demonstrate both front-end development ability and practical interactive features that reflect real-world software development and IT support experience.
